产品页面批量转换工具 · AEC GROUP

把所有"手写死数据"的旧版产品页，批量转换成"从 products-data.js 动态读取"的新版页面 —— 转换后后台改产品，前台所有页面都会自动同步。
用法和后台管理系统一样：连接你的 aec-project 文件夹，工具会在你的浏览器本地读写文件，不会上传到任何服务器。
1连接网站文件夹

点击下面按钮，选择你的 aec-project 文件夹（和后台"文件管理"里选的是同一个文件夹）。

2扫描页面

工具会检查文件夹里每一个 .html 文件，判断它是"旧版静态页面"还是"新版动态页面"，非产品详情页（如首页、联系页、后台文件）会自动跳过。

3确认要转换的页面
旧版待转换 0 已是新版 0 已跳过（非产品页） 0
全选 / 取消全选（仅"旧版待转换"可勾选）
文件名状态匹配的产品
⚠️ 转换前建议先备份 aec-project 文件夹（复制一份即可）。工具只会改写被勾选、且判定为"旧版"的文件，文件名不会改变（保证和后台产品数据里的 detailLink 继续匹配），其余文件不会被触碰。
